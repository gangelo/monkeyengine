# MonkeyEngine

Have some fun! MonkeyEngine is a gem that allows virtual monkeys to tap away on a virtual keyboard. 

## Installation

Add this line to your application's Gemfile:

    gem 'MonkeyEngine'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install MonkeyEngine
